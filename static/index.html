§ BetterWebUI

A thoughtful workspace for AI

    A workspace bundles a system prompt, chosen skills, MCP servers, CLI shortcuts, and persistent files into a saved configuration you can return to.

      Skills are short markdown briefs telling the assistant how to do something. The assistant loads one when the request matches.

        System prompts shape the assistant's role and tone. Pick the one to use as the active voice.

          MCP servers

          MCP servers extend the assistant with new tools — file systems, GitHub, web fetch, search, memory. Some require Node.js or Python's uv to be installed.

            CLI shortcuts

            Named command-line tools the assistant knows are available. The assistant calls them with arguments via cli_call; each invocation still needs your approval.
